# ehRadio Remote — PWA

غلاف PWA يحتضن واجهة ويب الراديو الحالية (ehRadio WebUI) داخل تطبيق قابل للتثبيت على الشاشة الرئيسية، بدون شريط عنوان المتصفح.

## الملفات
- `index.html` — الواجهة (شاشة إعداد IP + شاشة التحكم)
- `manifest.json` — بيانات التطبيق (الاسم، الأيقونة، اللون)
- `sw.js` — service worker يخزّن هيكل التطبيق فقط
- `icons/` — أيقونتا 192px و512px

## التشغيل والتثبيت على الجوال
تسجيل الـ service worker (وبالتالي إمكانية "التثبيت" الحقيقي كـ PWA) يتطلب تحميل الملفات عبر `http(s)`، وليس فتحها مباشرة كملف. أسهل طريقة:

1. انسخ مجلد `ehradio-pwa` كاملًا إلى أي جهاز على نفس شبكتك (الحاسوب مثلًا).
2. من ذلك الجهاز، شغّل خادمًا محليًا بسيطًا داخل المجلد:
   ```bash
   cd ehradio-pwa
   python3 -m http.server 8080
   ```
3. من متصفح الجوال (نفس الشبكة)، افتح:
   ```
   http://<IP-الحاسوب>:8080
   ```
4. من قائمة المتصفح: **"إضافة إلى الشاشة الرئيسية" / Add to Home Screen**.
5. عند أول فتح، أدخل IP جهاز الراديو نفسه (مختلف عن IP الحاسوب).

## ملاحظة عن الوصول للراديو
إذا استضفت هذه الصفحة لاحقًا عبر HTTPS على الإنترنت، قد يمنع المتصفح تحميل واجهة الراديو (HTTP) داخلها كسياسة أمان (Mixed Content / Private Network Access). للاستخدام المنزلي العادي عبر خادم محلي كما بالأعلى، هذا لا يشكّل مشكلة.

## التطوير لاحقًا
حاليًا الشاشة الرئيسية تعرض واجهة الراديو الأصلية كما هي داخل إطار مصمم. لو حبيت أزرار تحكم أصلية (تشغيل/إيقاف/تغيير محطة) بدل الإطار، أحتاج معرفة نقاط الـ API الفعلية (REST) أو تفعيل الاتصال عبر MQTT من داخل التطبيق مباشرة.
